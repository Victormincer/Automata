Este proyecto consiste en una aplicación desarrollada en Java para el estudio, simulación y manipulación de autómatas finitos. El programa permite crear, representar y analizar diferentes tipos de autómatas, incluyendo autómatas finitos determinísticos (AFD) y no determinísticos (AFND), además de realizar procesos como conversión entre modelos, validación de cadenas y visualización gráfica mediante diagramas de estados.

La aplicación está orientada al aprendizaje académico y práctico de la teoría de lenguajes formales y compiladores, proporcionando herramientas para comprender el funcionamiento interno de los autómatas y las expresiones regulares.

Características principales:

Creación de autómatas mediante estados y transiciones. Validación de cadenas de entrada. Conversión de AFND a AFD. Generación y visualización de diagramas de estados. Soporte para expresiones regulares y análisis léxico básico. Interfaz desarrollada completamente en Java.
